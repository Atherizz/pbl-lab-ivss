# Sistem Informasi Laboratorium IVSS

Sistem manajemen laboratorium berbasis web untuk Laboratorium Intelligent Vision and Smart System (IVSS) Politeknik Negeri Malang.

## Fitur Utama

1. **Smart Research & Publication Hub** - Pengelolaan riset dan publikasi secara terintegrasi
2. **Role-Based Access Control** - Multi-role user (Admin Lab, Admin Berita, Anggota Lab, Mahasiswa)
3. **Digital Approval Workflow** - Proses persetujuan penelitian dan peminjaman alat secara digital
4. **Google Scholar Integration** - Sinkronisasi publikasi dosen secara otomatis
5. **AI Supervisor Recommendation** - Rekomendasi dosen pembimbing menggunakan embedding vector search

## Tech Stack

- **Backend**: PHP 8.2, PostgreSQL, Custom MVC
- **Frontend**: Tailwind CSS, Alpine.js, FontAwesome
- **Infrastructure**: Docker, Nginx, SSL/HTTPS
- **AI**: Embedding vector search dengan OpenAI
